## O que é o método 50-30-20?

O método 50-30-20 é uma das formas mais simples de organizar o dinheiro. A ideia é dividir sua renda em três partes:

- **50%** para necessidades (aluguel, alimentação, transporte, contas)
- **30%** para desejos (lazer, restaurantes, streaming, roupas)
- **20%** para investimentos e dívidas

Parece simples. Mas será que funciona para quem ganha um salário mínimo?

## A realidade de quem ganha salário mínimo no Brasil

Em 2026, o salário mínimo é de R$ 1.518. Aplicando o método:

- **50% para necessidades** = R$ 759
- **30% para desejos** = R$ 455
- **20% para investimentos** = R$ 303

O problema é que para a maioria dos brasileiros que ganham salário mínimo, só o aluguel já consome mais de 50% da renda. Isso torna o método impossível de seguir à risca.

## Como adaptar o método para a realidade brasileira

A boa notícia é que o método pode ser adaptado. Em vez de 50-30-20, tente começar com:

- **70% para necessidades** — aceite que sua realidade é diferente
- **10% para desejos** — você merece algum lazer
- **20% para investimentos** — mantenha essa fatia sagrada

O mais importante não é seguir os percentuais exatos, mas criar o hábito de separar uma parte da renda antes de gastar.

## O que fazer quando as necessidades consomem tudo?

Se suas necessidades consomem mais de 80% da renda, o problema não é de organização — é de renda. Nesse caso, o foco deve ser:

1. **Reduzir custos fixos** — renegociar aluguel, trocar de plano de celular, cortar serviços desnecessários
2. **Aumentar a renda** — freelas, bicos, vender produtos
3. **Eliminar dívidas** — juros são o maior inimigo de quem ganha pouco

## Vale a pena tentar?

Sim, mesmo que você não consiga seguir os percentuais exatos. O método 50-30-20 é útil porque te faz **pensar antes de gastar**. E esse hábito, por si só, já transforma sua relação com o dinheiro.

Comece com o que você tem. Qualquer progresso é progresso.
